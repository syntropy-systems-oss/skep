## 0.0.0 (unreleased)

Initial scaffold — the cell/action runtime built on one abstraction, the bee:

- One actor (the bee), one model seam (`Mind.decide`, a policy: observation → action),
  one permission gate (capability strings), one value that crosses a boundary (the result).
  `resolve` is injected into every cell.
- `skep({ cells, mind, renderer? }).run(prompt)` weaves a comb.
- `llmMind()` — a per-instance-configurable, OpenAI-compatible policy in the agent layer;
  the core knows nothing about LLMs and has zero runtime dependencies.
- Safe-by-default `xml` markup (`raw()` to opt out), TUI/briefing renderers, and a debug
  TUI over the run event stream.
- Ships `dist` (JS + `.d.ts`) with a consumer-perspective typecheck and a `node:test` suite.
- Apache-2.0, with a runnable code-browser example and a hardened example Dockerfile.
